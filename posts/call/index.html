December 4, 2019

js基础知识—call,apply,bind的用法

应用举例

/*call*/
function add(a,b){
    return a + this.b;
}
let obj = {"b":100}

add.call(obj,3,2) // 结果：103

call的作用是指定函数体内部this的指向；call()方法中传入的第一个参数代表的就是this的指向。 上述代码中，add函数中的this指向obj，那么this.b就是obj.b,因此add函数输出的结果为3+100=103。

apply和call的作用完全一样，唯一不同的是他们接受的参数不同。apply接受数组参数，call接受连续参数。

/*apply*/
function add(a,b){
    return a + b;
}

add.call(add,3,5) // 8
add.apply(add,[3,5]) //8

bind接受的参数和call一样，它的作用同样是指定函数体内this的指向。 bind与call的区别是，bind()方法不会立刻调用，它会创建一个新的函数，称为绑定函数，可以在需要调用的时候再执行函数，并非是立即执行函数；而call，apply是在改变了上下文中的this指向后并立即执行函数。

/*bind*/
function showColor(){
    console.log("my color is:" + this.color);
}
window.color = "red";
showColor.bind(window); // 此刻不会打印结果

showColor.bind(window)() // 可以创建函数后直接调用
let color1 = showColor.bind(window);
color1() // 也可以将指定this的showColor函数赋值给新的变量color1,再执行。
