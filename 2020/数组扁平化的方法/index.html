数组扁平化的方法

Written by 周啟尧 with ♥ on April 16, 2020 in JavaScript

数组扁平化

将数组[1,[2,3,[4,[5]]]]变成[1,2,3,4,5]就是数组扁平化。

方法一 es6 的新 api flat(depth)

depth 默认是 1。

let a = [1, [2, 3, [4, [5]]]];
a.flat(); // [1, 2, 3, Array(2)]
a.flat(4 - 1); // [1, 2, 3, 4, 5]

方法二 for 循环+concat

let a = [1, [2, 3, [4, [5]]]];
function flatten(array) {
  let result = [];
  for (let i = 0; i < array.length; i++) {
    if (Array.isArray(array[i])) {
      result = result.concat(flatten(array[i]));
    } else {
      result.push(array[i]);
    }
  }
  return result;
}
