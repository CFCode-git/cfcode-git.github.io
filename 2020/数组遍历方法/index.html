列举数组遍历的方法

Written by 周啟尧 with ♥ on April 25, 2020 in JavaScript

for

let array = [1, 2, 3, 4, 5];
for (let i = 0; i < array.length; i++) {
  console.log(array[i]);
}

forEach

无返回值(返回 undefined)，会为数组的每一个元素执行一个 callback 函数。

let array = [1, 2, 3, 4, 5];
array.forEach((item) => {
  console.log(item);
});

for in

一般用来遍历对象的属性。

只遍历可枚举属性(enumerable)【包括非整数类型的名称和原型链上面的属性】

var obj = {
  name: "chow",
  color: "yellow",
  day: "Sunday",
  number: 233,
};
for (var key in obj) {
  console.log(obj[key]);
}

for of

遍历可迭代（iterable）对象

var arr = [{ name: "hello" }, 666, "test"];
for (item of arr) {
  console.log(item);
}

for in 与 for of 区别

for in 以原始插入顺序迭代对象的可枚举属性。包括继承链的对象属性。

for of 只遍历可迭代对象的数据。

map【返回新数组】

其实可以视为一个有返回值的 forEach，返回一个新数组。

let array = [1, 2, 3, 4, 5];
let array2 = array.map((item) => {
  item + 1;
});

reduce

对数组中的前后项做某种运算，累计最终值。

let array = [1, 2, 3, 4];
let sum = array.reduce(function (pre, next) {
  return pre + next;
}, 0);
// 0 是初始值

filter【得到新数组】

通过回调函数返回 true/false 过滤数组。

let array = [2, 3, 4, 5, 6];
let moreThan4 = arr.filter(function (number) {
  return number > 4;
});

every & some

every： 数组中所有值为 true 则返回 true，只要有一个为 false 返回 false。

some：数组中有一个值为 true 则返回 true，否则返回 false。

var arr = [1, 2, 3, 4, 5];
let result1 = arr.every(function (item, index) {
  return item > 0;
});

let result2 = arr.some(function (item, index) {
  return item > 4;
});
