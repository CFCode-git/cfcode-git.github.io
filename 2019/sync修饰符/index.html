如何理解Vue的.Sync修饰符

Written by 周啟尧 with ♥ on December 20, 2019 in 技术

有时候我们可能需要对一个prop进行双向绑定，但真正的双向绑定会带来维护上的问题。

因为子组件可以修改父组件，且在子组件和父组件都没有明显的改动来源。

因此Vue推荐以update:myPropName的模式触发事件取代双向绑定。

具体做法就是：首先在子组件中用$emit传递参数。
“update:title”是事件名，newTitle是传递给父组件的参数。

// 子组件
this.$emit('update:title',newTitle)

然后在父组件监听”update:title”事件，根据需要更新父组件上的数据属性。

// 父组件
<text-document
    v-bind:title="doc.title"
    v-on:update:title="doc.title=$event">
</text-document>

上面的模式很常见，于是Vue给这种模式提供给了缩写，就是.sync修饰符。

// 父组件
<text-document :title.sync="doc.title"></text-document>

// 相当于
//<text-document
//    v-bind:title="doc.title"
//    v-on:update:title="doc.title=$event">
//</text-document>

当我们需要在子组件上修改props传递的数据时，不能直接修改，而是要通过.sync修饰符，通知父组件修改。

案例

比如设想一个这样的场景：爸爸给儿子钱，儿子要花钱。

父亲组件
首先，爸爸要告诉儿子自己有多少钱（用v-bind:），同时爸爸要儿子在花完钱之后(v-on:‘儿子花钱事件’)，把余额还给自己(用$event)

<template>
    <div class="father">
        我现在有 {{fatherMoney}}
    </div>
    <hr />
    <Child :money="fatherMoney" v-on:update:money="fatherMoney = $event" />
</template>

<script>
import Child from "./Child.vue";
export default{
    data(){
        return {fatherMoney:10000};
    },
    components:{Child:Child}
};
</script>

<style>
.father {
    border:3px solid red;
    padding:10px;
}
</style>
  • <Child :money="fatherMoney" v-on:update:money="fatherMoney = $event" />
    意思是，给Child组件传递一个money，这个money的值就是fatherMoney，然后我要监听Child组件的’update:money’事件，如果事件触发，我就将Child传回来的值用’$event’拿到，更新给我的fatherMoney。

儿子组件
儿子要拿到爸爸的钱(用props)，并在花完钱之后还给爸爸(用$emit)

<template>
    <div class="child">
        {{money}}
        <button @click="$emit('update:money',money-100">
        <span>花钱</span>
    </div>
</template>

<script>
export default{
    props:["money"]
};
</script>

<style>
.child{
    border:3px solid green;
}
</style>
  • <button @click="$emit('update:money',money-100">
    意思就是，儿子在花钱之后(点击花钱按钮)，会触发一个叫”update:money”的事件（告诉爸爸我花钱了），并且把余额还给爸爸（money-100）

现在这个场景就实现了点击Child组件上的花钱按钮，然后father上面的钱就会同步减少，而Child也知道了father的钱减少。

简写爸爸组件
现在爸爸的代码的template部分可以用修饰符.sync简写:

<template>
    <div class="father">
        我现在有 {{fatherMoney}}
    </div>
    <hr />
    <Child :money.sync="fatherMoney" />
</template>

总结

<Child :money.sync="fatherMoney" />
// 等于
<Child :money="fatherMoney" v-on:update:money="fatherMoney = $event" />

其他：Vue的三条规则

  1. 组件不能修改props外部数据。
  2. $emit可以触发事件，并传参。
  3. $event可以获取$emit传递的参数。
